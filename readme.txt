This is a simple plugin that sends email notification for every new post submitted for review.

== Description ==
### Admin Post Notifier

Always want to be notified when any of your blog contributors submit a new post? If yes, the gaol of Admin Post Notifier is to deliver emails to the site admin when there is a new (or updated) pending post for review. 
No addition setting is required of you. The plugin utilizes the WordPress core email functionality to send email notifications when activated. 

== Frequently Asked Questions ==
= Is Admin Post Notifier free? =
Yes! 

= How can I make it work? =
You simply have to download the plugin and activate it. The plguin does the rest.

= What does this plugin do? =
Admin Post Notifier notifies the blog's admin via email when a user with a contributor role submits a post. 

= Can I change the email address the plugin sends notifications? =
No. The plugin only sends to the blog administrator's email address. You can only change the email address when you change the admin's email.
